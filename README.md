### Christmas Tree
	在樹上加上隨機顏色的燈，有一般模式(每0.5s同時亮起同時關閉)和複雜模式(每個燈隨機時間、隨機亮起)
	⚠️若沒關閉原本模式直接切換，會需要點及兩次才有反應

### Vector3
	輸入V和W後，輸出VW的內積和外積，需輸入完整才有反應

### 室內設計模型
